# 贡献指南

感谢你对本项目的兴趣！

## 如何贡献

1. Fork 这个仓库
2. 创建新分支 (`git checkout -b feature/功能名称`)
3. 提交更改 (`git commit -m '添加新功能'`)
4. 推送分支 (`git push origin feature/功能名称`)
5. 创建 Pull Request

## 贡献 ideas

- 新增 VPS 管理脚本
- 性能优化脚本
- 安全加固工具
- 监控模板
- 文档改进

## 指南

- 遵循现有代码风格
- 提交前测试脚本
- 必要时更新文档
- 保持提交原子性和良好描述
